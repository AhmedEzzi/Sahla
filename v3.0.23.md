# Changelog


## [3.0.23] -

### Added
- **Android Operations Support**
  - All Operations features are now accessible and usable on Android devices
  - Reduced dependency on dedicated PDA hardware

- **Barcode Mode (Android)**
  - New Barcode Mode available under Operations
  - Supported modules:
    - Items
    - Packages
    - Store Picking
    - Transfers
    - Batch Transfers
  - Enables barcode-based workflows without RFID hardware

- **Android Barcode-Based Operations**
  - Store Picking via barcode scanning
  - Transfers & Batch Transfers via barcode scanning
  - Open Package via barcode scanning only

- **RFID Validation Enhancements**
  - Detection of RFID tags not belonging to the current:
    - Package
    - RFID Transfer
    - Batch Transfer
  - Warning dialog displaying the number of unmatched RFID tags
  - New **Find Tag** feature to identify and locate specific unmatched tags

### Changed
- Clear separation between **Barcode Mode** (Android) and **RFID workflows** (PDA)

### Notes
- RFID operations require PDA devices
- RFID scanning is not supported on Android in this release
